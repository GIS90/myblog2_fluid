Python模块之企业微信消息

名称 版本
Python 3

描述

企业微信消息通知
腾讯企业微信官网提供一整套WebHook API接口，内容相当丰富，可以实现内部、第三方等各种各样的功能
详情参考reference urls

设计

运用requests模拟API请求，这里主要实现了2大请求：

  • ACCESS TOKEN：认证token获取
  • SEND MESSAGE：发送消息

使用

初始化必须的参数：

  • corp_id: 企业号标识
  • secret: 管理组凭证密钥
  • agent_id: 机器人应用ID
